# Double Elimination Bracket Generator

A web-based tournament bracket generator with parallel game scheduling.

## Features

- Add unlimited teams with custom names
- Automatic double elimination bracket generation
- Visual representation of Winners, Losers, and Finals brackets
- Parallel game schedule showing which games can be played simultaneously
- Responsive design with modern UI

## How to Use

1. Open `index.html` in your web browser
2. Add team names using the input field
3. Click "Generate Bracket" to create the tournament structure
4. View the bracket and parallel game schedule

## Bracket Structure

- **Winners Bracket**: Teams compete, losers drop to losers bracket
- **Losers Bracket**: Teams get a second chance
- **Finals**: Winner of each bracket competes for championship
- **Grand Finals**: If losers bracket winner wins, a second match determines champion

## Parallel Scheduling

The schedule shows which games can be played at the same time to minimize tournament duration.
